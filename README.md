## Setup & Run Locally

1️⃣ **Clone the repository**
```sh
git clone https://github.com/DavidMandado/MyPortfolio.git
```

2️⃣ Navigate to the project folder

```sh
cd MyPortfolio
```


3️⃣ Open index.html in your browser or use a local server like:

```sh
npx serve
```
